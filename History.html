JavaScript Today

Runs on browsers, servers (Node.js), mobile apps, desktop apps, IoT devices, etc.

Integral to frontend, and increasingly, backend and full-stack development.

Compare VAR, LET, CONST

var is function scoped

  • let and const are block scoped

var is hoisted and initialized with undefined

  • let and const are hoisted but remain in temporal dead zone until declaration

const means you cannot reassign the variable itself, but you can mutate the object/array it points to.
